# WM KI Predictor 2026

Überarbeitete React-Webseite auf Basis deines ursprünglichen Codes.

## Was verbessert wurde

- sauberere Struktur
- bessere mobile Darstellung
- robustere Teamsuche mit Umlauten
- Footer ergänzt
- Meta-Title, Meta-Description und Favicon ergänzt
- kein Tailwind nötig, dadurch einfacher zu starten
- direkt als Vite-Projekt vorbereitet

## Projekt lokal starten

```bash
npm install
npm run dev
```

## Für den Livegang

```bash
npm install
npm run build
```

Danach kannst du das Projekt z. B. bei Vercel oder Netlify hochladen.

## Wichtiger Hinweis

Impressum, Datenschutz und Kontakt sind aktuell nur Platzhalter-Links.
Bevor die Seite öffentlich online geht, solltest du diese Seiten richtig ergänzen.

## Später als App

Die Webseite kann später relativ einfach weiterentwickelt werden zu:

- einer PWA
- oder einer echten App mit React Native / Expo
